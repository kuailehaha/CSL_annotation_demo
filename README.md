# CSL_annotation_demo
这是一个中文标准/自然手语标注工具demo，使用gradio搭建。

## 功能
基于Qwen3-embedding-8B的向量化，作为标注工具向听障人士开放，并可用于查询，解决了手语数据集的同义词标注难点。
手语动作视频自动加载，添加一键复制功能，简化标注流程。
